# 🌸 AI Aesthetic To-Do App ✨

Welcome to the **AI Aesthetic To-Do App** — a modern productivity web application designed with a beautiful glassmorphism interface and smart task management features. 🚀💖

Stay organized, productive, and motivated while managing your daily tasks in style! 🌈📝

---

# 🚀 Features

✨ **Beautiful Glassmorphism UI**
Enjoy a modern aesthetic interface with smooth animations and elegant design.

🌙 **Dark / Light Mode**
Switch between themes for a comfortable experience anytime.

📅 **Task Management**
Add, complete, search, and delete tasks easily.

📊 **Progress Tracking**
Track your productivity with a dynamic progress bar.

🔍 **Instant Search**
Quickly find tasks using the smart search feature.

💾 **Local Storage Support**
Tasks are automatically saved in your browser.

⚡ **Responsive Design**
Works smoothly across desktop, tablet, and mobile devices.

⌨️ **Keyboard Support**
Press Enter to instantly add tasks.

✨ **Motivational Quotes**
Stay inspired with productivity quotes.

---

# 🛠️ Technologies Used

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript
* 💾 Local Storage API

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/samreen-08/cloud-todo-app.git
```

## Navigate to the Project Directory

```bash
cd cloud-todo-app
```

## Run the Project

Simply open:

```bash
index.html
```

in your browser 🌸
